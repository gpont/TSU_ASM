# TSU Assembler labs
Tomsk State University, Faculty of Applied Mathematics and Cybernetics, 2 year, 2 semester, Assembler

## Compiling and running
```
make <task_name>
```
Tasks:
 - `chess`
 - `int`
 - `progression`
 - `square`
 - `invert`
